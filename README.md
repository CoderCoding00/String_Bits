# Sting Bits Instructions  
```md 
Given a string, set result to a new string made of every other char
starting with the first, so "Hello" yields "Hlo".

Output
• for input of "Hello" → "Hlo"
• for input of "Hi" → "H"
• for input of "Heeololeo" → "Hello"
```
